Logiciels utilisés

Git

Git est un logiciel qui nous permettra de se partager le code commun tout en faisant notre propre code chacun de notre côté donnant la possibilité de collaborer en gardant une autonomie et une indépendance de chacun des membres, évitant donc un arrêt total du développement du jeu en cas de problèmes chez un des membres.


Pixel Studio

Pixel Studio qui est simple à prendre en main, mais complet pour le projet, cela permettra de faire toutes les animations du jeu allant de notre personnage principal au boss final en passant par les monstres et membres du village.


Unity

Unity sera utilisé afin de regrouper l'entièreté du projet, il en est le point central où convergeront codes, audios, fichiers design, etc.


FL Studio

Les musiques du jeu seront à la fois récupérées en partie sur Internet ainsi que créer via ce logiciel.

Images

Nous avons utiliser des divers assets visuels pour la conception de notre jeu vidéo.


Pour les blocs qui constituent notre monde, nous avons utiliser des asssets de https://itch.io/game-assets/tag-parallax.
